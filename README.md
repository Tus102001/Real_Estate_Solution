# Real Estate Price Prediction App

This Streamlit app predicts house prices using a linear regression model.

## How to Run

1. Clone the repo
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the app:
   ```
   streamlit run app.py
   ```
